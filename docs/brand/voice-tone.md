# Voice & Tone

> How Flowe communicates with visitors and users

## Brand Personality

If Flowe were a person:
- **Age:** 28-35 years old
- **Tone:** Friend who understands you, not a coach who pressures you
- **Attitude:** Patient, never judges
- **Style:** Minimalist but warm
- **Voice:** Calm, direct, positive

## Flowe IS:

- Calm
- Understanding
- Realistic optimist
- Professional but approachable
- Modern
- Inclusive

## Flowe IS NOT:

- Clinical/medical
- Childish
- Condescending
- Rigid
- Blame-inducing
- Cold/corporate

## Communication Principles

### 1. Positive, Not Condescending
```
"You completed 3 tasks today!"
NOT: "You only completed 3 tasks"
NOT: "Wow, incredible, you did it!" (exaggerated)
```

### 2. Direct, Not Cold
```
"Join the waitlist"
NOT: "Submit your email address for waitlist registration"
NOT: "Submit."
```

### 3. Empathetic, Not Clinical
```
"Built for how your brain actually works"
NOT: "Designed for executive dysfunction management"
NOT: "ADHD disorder treatment tool"
```

### 4. Guide, Don't Command
```
"Ready to find your flow?"
NOT: "Sign up now before it's too late"
NOT: "You need this app."
```

## Words to Use

| Use | Instead of |
|-----|------------|
| Complete | Finish/end |
| Pause | Stop |
| Adjust | Change/modify |
| Today | Immediately |
| Progress | Performance |
| Plan | Agenda/schedule |
| Flow | Productivity |

## Forbidden Words

Never use these in website copy:

| Word | Reason |
|------|--------|
| Fail / Failed | Shame-inducing |
| Late / Behind | Anxiety-inducing |
| Must / Have to | Commanding |
| Urgent | Pressure |
| Incomplete | Negative framing |
| Deficit | Clinical |
| Disorder | Medical |
| Symptoms | Clinical |
| Limited time | Pressure tactic |

## Website Copy Tone

### Hero Section
- Warm, inviting, confident
- "Productivity that flows with you"
- NOT "The #1 ADHD productivity solution!"

### Features Section
- Clear, benefit-focused
- "See your day at a glance"
- NOT "Advanced timeline visualization system"

### Waitlist CTA
- Gentle invitation
- "Join the waitlist"
- NOT "Don't miss out! Sign up NOW!"

### Footer / Legal
- Professional, transparent
- Keep it clean and simple

## Emoji Usage

- Maximum 1-2 per section
- Use sparingly on the website
- Checkmark is acceptable
- Avoid faces and excessive celebration

## Website-Specific Copy Examples

```
// Hero
"Your rhythm, your flow"
"The productivity app that works with your ADHD brain, not against it."

// Feature highlight
"No guilt trips. Just progress."
"Missed a task? No problem. 3 of 5 done."

// Waitlist CTA
"Be the first to flow"
"Join 2,000+ people waiting for a calmer way to plan"

// Social proof
"Built by people who understand ADHD"
```
