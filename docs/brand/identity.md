# Brand Identity

> Flowe visual identity guidelines

## Name

**Flowe**

Pronunciation: /flow-ee/ (flow-ee)

### Meaning
- **Flow:** Fluidity, mental flow state
- **e:** The macron (line over e) represents a gentle wave -- frictionless movement

## Tagline

> "Your rhythm, your flow"

Alternatives:
- "Flow your way"
- "Productivity that flows"
- "Flow with you"

## Logo

### Primary Logo
```
  flowe

  The macron line (e) represents
  a gentle wave -- continuous flow
```

### Construction
- **Typography:** Based on Lexend or Inter, modified
- **Weight:** Medium or SemiBold
- **Tracking:** Slightly expanded
- **The e macron extends subtly as a wave**

## Logo Variants

| Variant | Use |
|---------|-----|
| **Full logo** | Website header, marketing materials |
| **Symbol only** | Favicon, social media avatar |
| **White logo** | Dark backgrounds |
| **Dark logo** | Light backgrounds |

## Clear Space

Minimum space = height of "o" on all sides

## Minimum Size

- **Digital:** 24px height
- **Print:** 10mm height

## Incorrect Usage

- Don't stretch or compress
- Don't rotate
- Don't change colors outside palette
- Don't add effects (shadows, gradients on the logo itself)
- Don't use on low-contrast backgrounds
