# Brand Assets

> Guidelines for website and marketing materials

## Favicon

```
Sizes: 16x16, 32x32 (.ico), scalable (.svg)
Background: Primary (#5B8FB9)
Symbol: White wave / stylized e
```

## Open Graph Image

```
Size: 1200 x 630px
Background: #EDF1F5
Logo: Centered, primary color (#5B8FB9)
Tagline: Below logo
Format: PNG
```

## Apple Touch Icon

```
Size: 180x180px
Background: Primary (#5B8FB9)
Symbol: White wave
```

## Website Screenshots

### Style Guidelines
- Background: `#EDF1F5` (neumorphic)
- Device: Clean mockup of the mobile app
- Text: Lexend, max 5 words per slide
- One feature per image

### Required Images
| # | Feature | Alt Text |
|---|---------|----------|
| 1 | Timer | "Visual timer for focus" |
| 2 | Timeline | "See your day at a glance" |
| 3 | Tasks | "Simple task management" |
| 4 | Progress | "Celebrate your wins" |
| 5 | Flexibility | "Adapts to your day" |

## Social Media

### Handles
| Platform | Handle |
|----------|--------|
| Instagram | @floweapp |
| Twitter/X | @floweapp |
| TikTok | @floweapp |

### Post Style
- Use brand colors as backgrounds
- Lexend typography
- Short, positive messages
- Maximum 1-2 emojis
- No excessive punctuation

### Content Themes
- ADHD tips and understanding
- Feature highlights
- Motivational (not toxic positivity)
- Behind-the-scenes development

## Brand Checklist

Before publishing any material:

- [ ] Uses colors from official palette?
- [ ] Typography is Lexend?
- [ ] Tone is positive and empathetic?
- [ ] Avoids forbidden words?
- [ ] Logo has clear space?
- [ ] Animations are subtle?
- [ ] Contrast is adequate (4.5:1)?
- [ ] Feels calm, not overwhelming?
- [ ] Responsive on all devices?
